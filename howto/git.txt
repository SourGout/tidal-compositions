Initialize a git repository:
git init

Status
git status

Add (ALL) files to repository
git add .
OR
git add ice.tidal

Create a commit
git commit -m "Init commit"
(-m "" is the message describing your changes)

See log of commits (changes)
git log

Made changes you didn't like? Checkout to previous commits

HEY I CHANGED THINGS AND I WANT THE CHANGES ON THE INTERNET
git status (WHATS UP?)
git add . (ADD ALL THE FILES) OR git add ice.tidal 
git commit -m "CHANGES I WANT ON THE INTERNET"
git push

GET SOMETHING FROM THE INTERNET
git clone THETHING (make sure ssh, not https)
git clone git@github.com:paullucas/supercollider-works.git
